# Analisis Kepuasan Pengguna TikTok Shop (Metode Cochran)

Proyek ini bertujuan untuk mengukur tingkat kepuasan mahasiswa terhadap layanan TikTok Shop menggunakan analisis statistik di RStudio.

## Metodologi
- **Populasi:** Mahasiswa pengguna TikTok Shop (34.9%).
- **Sampel:** 57 Responden (Dihitung dengan rumus Cochran, MoE 12%).
- **Analisis:** Uji Validitas, Reliabilitas (Cronbach's Alpha), dan Statistik Deskriptif.

## Hasil Utama
- **Reliabilitas:** 0.877 (Sangat Tinggi).
- **Temuan:** Indikator kemudahan penggunaan memiliki skor tertinggi (4.04).

## Cara Menjalankan Code
1. Clone repositori ini.
2. Pastikan library `dplyr`, `psych`, dan `ggplot2` sudah terinstall.
3. Jalankan `script_analisis.R`.
